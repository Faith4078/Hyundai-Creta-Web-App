## Note

# For Name translation

For 150k users, you'll need a paid translation service. Here's the reality:
Cost Analysis:
Google Cloud Translation API:

First 500,000 characters/month: FREE
After that: $20 per 1 million characters
Average name ~10 characters = 1.5 million characters for 150k users
Cost: ~$20-30 one-time (if translating once during signup)

Microsoft Azure Translator:

2 million characters/month: FREE
After that: $10 per 1 million characters
Cost: $0-10/month (cheaper than Google)

DeepL API:

500,000 characters/month: $5.49
Then $25 per 1 million characters
More accurate than Google
Cost: ~$40-50
